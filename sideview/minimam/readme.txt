Character graphics for a turn based RPG. 4 PC classes, 2 monsters, 4 poses for each character (healthy, injured, incapacitated and attacking.) Sketches, vector graphics (SVG) and bitmaps (PNG) included, as well as some UI graphics (an escape button and circles to go under healing and attacking targets.)

Artists: Seth Galbraith (sketches, UI) and Ulrich Galbraith (vector graphics)

Interactive demo (python/pygame): http://code.google.com/p/minimam/
